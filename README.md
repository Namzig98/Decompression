# Decompression

A multiplayer sci-fi horror game built in s&box. Saboteurs hidden among the crew vent station sections by hacking control panels — get caught inside, get sucked into space.

## Project state

- Engine: s&box (C# / Sandbox.* APIs, scene + component system, multiplayer @ 50Hz)
- This is the v2 build. Group B (player & death systems) is the current implementation focus.
